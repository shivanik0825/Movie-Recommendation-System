# Movie Recommendation System 🎬

An interactive web application built with Python and Streamlit that recommends 5 similar movies based on a user's selection using Machine Learning.

## 🚀 Features
* Clean and responsive web interface using Streamlit.
* Vectorized movie dataset comparisons utilizing a pre-computed similarity matrix.
* Instant recommendations.

## 🛠️ How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/shivanik0825/Movie-Recommendation-System.git

2. Navigate into the project folder:
   ```bash
   cd Movie-Recommendation-System


3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt


4.Run the Streamlit application:
   ```bash
   streamlit run app.py
